# Contributing

Thanks for contributing to Telegram Addon Studio.

## How to add a new addon

1. Run `python scripts/new_addon.py <addon-name>`.
2. Build the smallest useful version first.
3. Add clear setup steps and config docs.
4. Update the root README catalog.
5. Update `docs/roadmap.md` if the addon changes the plan.

## Style

- Keep dependencies small
- Prefer readable code over clever code
- Document limits honestly
- Never commit secrets
